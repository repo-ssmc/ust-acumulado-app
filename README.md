# ust-acumulado-app

El aplicativo Acumulado UST ha sido desarrollado con el objetivo de automatizar la consolidación de los registros estadísticos mensuales (REM) de Producción Hospitalaria correspondientes a un mismo establecimiento, serie y mes. La herramienta suma automáticamente los valores contenidos de estas tres series:

  - Hospital Clínico Metropolitano El Carmen Doctor Luis Valentín Ferraday

  - Hospital Clínico San Borja Arriarán

  - Hospital de Urgencia Asistencia Pública Dr. Alejandro del Río

Y posterior a esto genera un único archivo consolidado, correspondiente a la Unidad de Salud del Trabajador (UST). De esta forma se reduce el trabajo manual, se evitan errores de digitación y se agiliza la entrega de la información consolidada.
